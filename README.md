# Nx workspace with Nuxt

This is a Nx workspace with a Nuxt app, a Vue app, and three Vue libraries. It was generated using [Nx](https://nx.dev).

## How to use

1. Install dependencies

```bash
npm install
```

2. Run the Nuxt app

```bash
nx serve my-nuxt-app
```

3. View the project graph

```bash
nx graph
```
